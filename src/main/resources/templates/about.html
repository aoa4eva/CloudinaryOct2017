About the Movie Database

This application was developed for the Java Bootcamp October 2017. It has the following features:

  1. Many to many relationships between movies and actors
  2. A form to add actors to the database
  3. A form to add movies to the database
  4. A form to add actors to movies
  5. Thymeleaf fragments, including:
    • A condition to display links only when actors have been enterd into the database AND there are still some actors to be added to the movie
    • A navigation bar that indicates where in the application a user is
  6. A method has been added to the Cloudinary configuration to transform images based on the featured actor's face. Have a look at this code as well!
    7. You will be given the code of this application to type out. Make sure you don't leave anything out!
